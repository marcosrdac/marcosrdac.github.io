Tipos de dados

A primeira coisa a ser entendida em uma linguagem de programação diz respeito aos tipos de dados básicos:

Números

  • int números inteiros (obs.: engloba o antigo tipo "long")
    exemplos: '1', '42' e '666'.
  • float números racionais (do inglês "floating point" - números "de ponto flutuante" - assim chamados por usarem notação matemática similar à científica, mas em base 2, que facilita cálculos de números fracionários com +- 15 casas decimais de precisão)
    exemplos: '1.2345', '6.2831853072' e '2.71848281825'.
  • complex números complexos
    exemplos: '2j', '3 + 4j' e '(5+6j)'.
